Welcome to HackOR Battle of the Scientists

You can download Hack Oregon campaign finance data here. The csv files are zipped up into two 50MB tarballs (so that GitHub doesn't complain):

Raw Committe Transactions table raw_committee_transactions.csv.gz
All other database tables all-tables-except-raw_committee_transactions.csv.gz

Sometimes ;) you can use a Django admin interface at totalgood.org/admin to browse the database and even perform RESTful GEt queries by hacking the URL. Our double-secret password is "admin", and I bet you can guess the username.

And the passwordless RESTFUL API for all the PAC (Political Action Comittee) tables is at totalgood.org/pacs
